Bible Nature Wallpaper App

This Android app automatically creates scenic nature wallpapers with Bible verses and updates your wallpaper about every 8 hours.

Download Android APK

If the button does not work yet, the APK has not been built by GitHub Actions. Go to your GitHub repository, open Actions, run Build Android APK, then return here.

Install steps

  • Open this page on your Android phone.
  • Tap Download Android APK.
  • Open the downloaded file.
  • Allow install from browser/files if Android asks.
  • Open the app once after installing so the automatic schedule starts.

Android may show a warning because this app is installed outside the Play Store. That is normal for APK downloads.
